# Microservices Architecture Demo

## Overview

This project demonstrates a microservices-based architecture built for learning purposes. The focus is on integrating multiple technologies within a containerized environment.

## Tech Stack

* **Frontend:** Svelte
* **Backend:** Microservices architecture
* **API Gateway:** Centralized request routing
* **Communication:**
  * REST
  * gRPC
  * Apache Kafka (event-driven messaging)
* **Authentication:** JWT
* **Containerization:** Docker & Docker Compose

## Key Concepts

* Microservices & distributed systems
* Event-driven architecture
* Hybrid communication (sync + async)
* API Gateway pattern
* Containerized deployment

## Run

```bash
docker-compose up --build
```
